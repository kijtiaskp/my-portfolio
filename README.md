Your project is live at:

**[https://kijtisakp.github.io/my-portfolio](https://kijtisakp.github.io/my-portfolio)**

## How It Works

1. Push changes to the `main` branch
2. GitHub Actions builds the Next.js static export
3. GitHub Pages deploys the latest version
